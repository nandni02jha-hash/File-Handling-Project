# 📂 File Handling Project (Python)

This is a simple command-line based file manager built using Python.
It allows users to perform basic file operations.

## 🚀 Features

* Create a file
* Read a file
* Update a file (rename, overwrite, append)
* Delete a file

## 🛠️ Technologies Used

* Python
* pathlib module
* os module

## ▶️ How to Run

1. Download the project
2. Open terminal in the project folder
3. Run:
   python main.py

## 💡 Purpose

This project was built to practice file handling and exception handling in Python.
